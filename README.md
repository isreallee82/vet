
## Overview

### Technologies

The project leverages:

- [UIkit](https://getuikit.com)
- [Pug](https://pugjs.org)
- [Less](http://lesscss.org)
- [Gulp](https://gulpjs.com)


## Quick start

### What's included

```
uikit-ecommerce-store/
├── src/
|   ├── scripts/
│   |   ├── script.js
|   ├── styles/
│   |   ├── components/
|   |   ├── style.less
|   ├── templates/
│   |   ├── components/
|   |   ├── data/
|   |   ├── layouts/
|   |   ├── mixins/
|   |   ├── pages/
|   |   |    ├── index.pug
|   |   ├── partials/
|   ├── images/

```

### Build

```bash

# Go into the repository
$ cd uikit-ecommerce-template

# Install dependencies
$ npm install

# Start
$ gulp
```
